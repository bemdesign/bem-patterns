# Бейдж
Используется для отображаения статуса. Чаще всего встречается в таблицах, для явной подписи состояния.

### pt-bage

| Модификатор | Значение                          | Описание                |  
| ----------- | --------------------------------- | ----------------------- |
| view        | success / warn / alert / normal   | Фон блока               |

## Пример
```
{
	block: 'bage',
	mods: { view: 'success' },
	content: 'success'
}
```
